## 📑 Description
    
**테이스팅 노트 기록**  
- 와인, 커피, 위스키 테이스팅 노트를 기록할 수 있습니다.
- 이름, 날짜, 사진, 색상, 향, 맛을 상세하게 기록할 수 있습니다.
- 작성한 테이스팅 노트를 모아 확인할 수 있습니다.
